# Simple Tasker Roadmap

## MVP (Phase 1)

Goal: basic task management via REST API

- Task entity (id, title, description, status)
- Task CRUD API
- Basic validation
- PostgreSQL integration
- Docker setup

## Phase 2

- User entity
- Authentication (JWT)
- User-task ownership

## Phase 3

- Projects
- Tasks grouping by project

## Phase 4

- Comments on tasks
- Task priority

## Future (optional)

- Notification service
- Kafka integration
- Microservices architecture
